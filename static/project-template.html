The Challenge

University scheduling is an NP-hard problem. Manually assigning classes to rooms while avoiding conflicts, considering professor availability, and managing lab resources usually takes weeks of effort.

The Solution

SmartFlex automates this entirely using Google OR-Tools. It treats the schedule as a constraint satisfaction problem. I built a Python backend that ingests CSV data of students, teachers, and rooms, and outputs a conflict-free mathematical solution in seconds.

Key Features

  • Constraint Satisfaction Logic (CSP)
  • Drag-and-Drop Adjustment UI
  • Real-time conflict detection
  • PDF Export for notice boards
